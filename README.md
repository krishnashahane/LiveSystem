# Live System Monitor ⚡

A **real-time terminal dashboard** that visualizes your system like a hacker-style control panel.

Monitor your machine’s performance directly from the terminal with a clean and powerful interface.

---

## 🚀 Features

* **CPU Monitoring**
  Real-time CPU usage and load tracking.

* **Memory (RAM) Usage**
  Live view of memory consumption and availability.

* **Network Traffic**
  Monitor incoming and outgoing network activity.

* **Running Processes**
  See active processes and resource usage.

* **GPU Usage**
  Track GPU utilization and performance (if supported).

* **Hacker-style Terminal Dashboard**
  Clean terminal UI inspired by futuristic system monitors.

## 📦 Installation

Clone the repository:

```
git clone https://github.com/yourusername/live-system-monitor.git
cd live-system-monitor
```

---

## ▶️ Run the Project

### Node.js version

```
npm install
node monitor.js
```

### Python version

```
pip install -r requirements.txt
python monitor.py
```
